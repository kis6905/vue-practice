## 개요
* vue js 연습!

## Requirements
* node(npm)
* vue-cli

## Build
``` bash
# git clone
git clone {this repository}

cd vue-practice

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
